Beautiful Designs Like Aphrodite, Strong Builds Like Zeus

AEGEAN is a refreshing, breezy and very effective CSS framework. It is a true reflection of the beauty of Aegean region which includes both Anatolia and Greece. It has been mainly inspired by the old Greek Mythology. Its design characteristic reflects those region's colors, foods, nature, weather and its rustic culture. This framework uses very light, pastel toned colors mainly based on weather and sea colors with specific fonts with minimalistis design approach.

Let's Start

AEGEAN is a very customizable and liberated framework. Designers, web developers or newly beginners can create their own customized designs over this modular CSS framework. It is a very light weight and user friendly framework that everybody can join a lot while building the small to big projects.

Please download the AEGEAN by clicking to Download button above. You will see a zipped folder, after unzipping it you will see 2 main folders. They are "css" and "scss" folders. Inside the css folder you will find the "aegean.css" file. Let's put this into your head tag in your html file.


There are 3 special fonts in AEGEAN, you can change them anytime or use them if you want to keep them. Main fonts are "GFS Diot" and "GFS Neohellen". There is another font type which is called "Mystery Quest" that you can use it for more funky, dynamic contents like about food, nightlife, adventure, etc.


AEGEAN In Details

AEGEAN has two main folders one of them is CSS and the other one is SCSS. AEGEAN is backed up by SCSS but at the same time your design elements are coming from CSS. In SCSS file you can be able to see the categorization titles for each components and additional elements, variables. With this way you will be able to interact with the objects of your website more easily. You will see different variables, navbar, cars, buttons, spacing, allignment, banner and grid information.


How Can I Change It?


No problem. If you want to make any single change, you can directly start to working on the variables in "variables.scss" file. Please do not forget to use preprocessor like Prepros in order to see the SCSS modifications synch properly with your CSS file.

If you do not prefer to interact with Aegean's main files and create your own CSS or SCSS, you can built them up in "customize.scss" file which is an empty file located in scss folder.

Sections

In order to creating a container perspective, put your code into "section" element by creating sections. With this way your contents will be more organized. Containers are very helpful in order to create a clean design in grid systems.




Card Boxes

In AEGEAN you can create these containers with cards. You can easily asjust the width and height of these card boxes and you can easily containerize your content in more organized way. You can also use the default colors (card-primary is one of these default colors of the cards) of cards component. We will look into Cards component later in this documentation.




Height or Width?

In cards.scss file you can easily change the width, heigh or other specifications in card-box variable.




Breakpoints

There are 4 default breakpoint sizes in AEGEAN. Smaall is 320px, medium is 480px, large is 960px and x-large is 1200px. You can easily change tehse values if you want to setup a specific media queries by simply change these values in "variables.scss" file under the Breakpoints section.



Fonts

In Aegean there are 3 kinds of modular type scale. One of them is perfect fourth (1:1.333), second one is major second (1:1.125) and the third one is augmented fourth (1:1.414). You can have more information about modular typhographic scale subject on this page. You can find these scales in "variables.scss" file. You can also check the details of these variables in "fonts.scss" file.




How To Change Size?

If you want to change the scale just simply open the fonts.scss file and find the variable called "$scale", you can assign any modular type scale value to this variable. You can also change the initial font size by putting any value to the "--initial-font" variable in same file.




Alignment & Spacing

Padding can be done with the variables in spacing.scss file. You can vertically or horizontally position your objects in your page by adding "padding-parallel" and "padding-up-and-down" classes in your elements. These classes are located on "spacing.scss" file. You can also see the different size spans in spacing.css. there are x-small, small, medium, large and x large sizes available in both vertically and horizontally.




There is also size mapping available in variable.scss file.




Text alignment

In order to aligning your text to right, to left or to center, you can add the "align-to-left", "align-to-right" and "align-to-center" classes into your elements. These variables are located in "alignment.scss" file.

Cards

In AEGEAN there are 3 types of cards. With cards class you can create little cards in your contents and put some important information into these cards. This is a great method to give a strong mention about these information. Card variables are located in "cards.scss" file. You can also see the card colors, margins and padding variables inside the "variables.scss" file.




Default

Default card has the smoothest color in the background it is more matching with the other colors.

Primary

Primary card has the brightest color. Use this card if you want to mention your text stronger.

Other

Other card has the most vibrant color. Use this card for more funky contents like food or nature.
